# Note-WebApp
This is a Note taking WebApp that allows users to Create, Edit and Delete notes.

Features:
Create new notes
Edit existing notes
Save notes
Delete Notes

Technologies used:
HTML
CSS
JavaScript

Instructions for use:
Clone the repository
Open the index.html file in a web browser
